### 运算符

#### 算术运算符

```java
package com.wangchaozi.arithmeticoperator;

public class ArithmeticoperatorDemo1 {
    public static void main(String[] args) {
        // +
        System.out.println(3 + 2);
        // -
        System.out.println(5 - 1);
        // *
        System.out.println(5 * 2);
        // /
        System.out.println(5 / 2);
    }
}

```
